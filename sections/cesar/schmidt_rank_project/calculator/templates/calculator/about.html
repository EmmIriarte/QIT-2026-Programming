About Schmidt Rank

What is Schmidt Rank?

The Schmidt rank is a fundamental measure in quantum information theory that quantifies the entanglement of a bipartite quantum state.

For a pure state |ψ⟩ in a bipartite Hilbert space H_A ⊗ H_B, the Schmidt decomposition is:

|ψ⟩ = Σᵢ λᵢ |iᴬ⟩ ⊗ |iᴮ⟩

Where:

  • λᵢ are Schmidt coefficients (non-negative real numbers)
  • |iᴬ⟩ and |iᴮ⟩ are orthonormal bases
  • Schmidt rank = number of non-zero λᵢ
Product States

Schmidt Rank = 1

State can be written as:

|ψ⟩ = |a⟩ ⊗ |b⟩

Examples:

  • |0⟩ ⊗ |0⟩
  • |+⟩ ⊗ |+⟩

NOT entangled

Entangled States

Schmidt Rank > 1

Cannot be written as product state

Examples:

  • Bell States (rank = 2)
  • |Φ+⟩ = (|00⟩ + |11⟩)/√2
  • Partially entangled: 0.8|00⟩ + 0.6|11⟩

ENTANGLED

Mathematical Method

This calculator uses Singular Value Decomposition (SVD):

  1. Reshape state vector into matrix form (dim_A × dim_B)
  2. Apply SVD: A = U Σ V†
  3. Schmidt coefficients = singular values (diagonal of Σ)
  4. Schmidt rank = number of non-zero singular values
Von Neumann Entropy

The entropy is calculated as:

S = -Σᵢ λᵢ² log₂(λᵢ²)
  • S = 0 for product states
  • S > 0 for entangled states
  • S = log₂(d) for maximally entangled states (d = dimension)
