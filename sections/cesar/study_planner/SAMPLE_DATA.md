## How to Import This Data

You can add this data through the Django admin panel:

1. **Log in to admin**: http://127.0.0.1:8000/admin/
2. **Add Programs** first (QIT and MMAD)
3. **Add Courses** (select the program for each)
4. **Create Enrollments** for yourself in courses you're taking
5. **Add Assignments** to track deadlines
6. **Record Grades** as you receive them

Or you can create a custom Django management command to import all data at once!
